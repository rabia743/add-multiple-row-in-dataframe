# Pandas: Adding Rows to a DataFrame 🐼

A simple Python project that demonstrates different ways to add new rows to a Pandas DataFrame.

## Features

* Create a DataFrame from a Python dictionary.
* Add single rows using `.loc`.
* Add multiple rows using `pd.concat()`.
* Display the updated DataFrame after each method.

## Methods Used

### 1. Using `.loc`

Adds one row at a time by inserting data at the next available index.

### 2. Using `pd.concat()`

Creates a new DataFrame containing multiple rows and combines it with the existing DataFrame.

## Technologies

* Python
* Pandas

## Learning Outcome

This project helps beginners understand different techniques for adding rows to a Pandas DataFrame and when to use each method.
